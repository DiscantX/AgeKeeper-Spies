# agekeeper-spies

Standalone Spies watcher runtime extracted from AgeKeeper.

## Install

```bash
pip install "agekeeper-spies @ git+https://github.com/<you-or-org>/agekeeper-spies.git@main"
```

## Run

```bash
agekeeper-spies
```

## Notes

- Depends on `agekeeper` for `lobby`, `shared`, and `aoe2api` modules.
- Windows-only runtime features are used for toast notifications/audio.
